Project Overview
Design and build a web application that displays a family tree.
Core Requirements:

Provide a search feature that allows users to search the Persons dataset using partial first or last names.
Display a list of matching person records based on the search input.
Allow the user to select a person from the search results.

On Selection:

Display detailed information for the selected person, including:

Full name
Parents
Spouse
Children
Additional Notes:
Ensure the UI is simple and user-friendly.
Structure the data to support relationships (parent, spouse, children) efficiently.

Tech Stack
Python
Flask
HTML

File Structure
app.py for the main application
Templates folder for HTML
Static folder for CSS
